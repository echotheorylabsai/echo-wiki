# Nader Dabit

Developer educator and builder in the AI agent tooling ecosystem.

## Relevance

Author of the tutorial "How to Build a Custom Agent Framework with PI" which walks through [[tools/pi-framework|PI framework]]'s layered architecture for building AI agents. The tutorial documents how PI's packages power [[tools/openclaw|OpenClaw]] and demonstrates building a persistent coding assistant in ~120 lines.

## Key Contributions

- Detailed walkthrough of PI's four-layer stack (pi-ai, pi-agent-core, pi-coding-agent, pi-tui)
- Documentation of production patterns from OpenClaw (multi-provider auth, stream middleware, event routing)
- Publishes on Substack covering agent frameworks and developer tooling
